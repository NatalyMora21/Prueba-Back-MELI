# Prueba-Back-MELI


## Info
API REST que permite las consultar un lista de productos y el detalle de cada uno


## Técnologías y herramientas ⚙️

* Java
* Spring Boot


## Enpoints 🚀


## Intems
**GET** /api/items/:query

*Trea todos los  productos relacionados a la búsqueda*
*-----*

## Detail Item
**GET** /api/item/:id

*Trae el detalle específico del producto*
*-----*
